## 0.1.0

- Initial release.
- Full DBC file parser with messages, signals, nodes, comments, and value descriptions.
- Standard and extended multiplexing (`M`, `m<N>`, `m<N>M`).
- Intel and Motorola byte order support.
- Signal compiler to native C structs for can_engine interop.
- Extended CAN ID (29-bit) detection.
- Validated against opendbc and j1939_dbc corpora.
